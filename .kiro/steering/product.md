# AfriSwap Web3 Conversion App

AfriSwap is a Web3 cryptocurrency conversion application focused on African markets. The app enables users to convert between various cryptocurrencies and traditional currencies with real-time exchange rates.

## Key Features

- **Cryptocurrency Conversion**: Convert between different crypto assets and fiat currencies
- **Real-time Exchange Rates**: Live pricing data with automatic updates
- **Wallet Integration**: Connect Web3 wallets (HashPack, Blade) for Hedera network
- **Transaction History**: Track and manage conversion history
- **Price Alerts**: Set notifications for favorable exchange rates
- **Bank Account Management**: Link traditional banking for fiat conversions
- **Admin Dashboard**: Administrative interface for platform management

## Target Market

The application is designed for African users seeking to bridge traditional finance with Web3 cryptocurrency ecosystems, providing accessible conversion tools with local market focus.

## Technical Context

Built as a modern React SPA with Supabase backend integration, featuring responsive design and dark/light theme support. The app emphasizes user experience with comprehensive UI components and real-time data synchronization.
